# MonochromeTV Music Search

## Overview
A web app for Google TV (and beyond) that lets users search music via monochrome.tf backend.  
Includes search history and caching. Can be wrapped as an Android/TV app or used standalone.

## Getting Started

### Backend
```bash
cd backend/
npm install
node server.js
```
